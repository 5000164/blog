## 設定

## Mac Modifier Keys

- Caps Lock: Control
- Conrtol: Control
- Option: Command
- Command: Option

## Mistel Barocco マクロ設定

- Left Control: Fn
- Left Command: Right Command
- Right Command: Left Command
- Backspace: \\\
- \\\: Backspace
- Fn + \\\: Delete
- Esc: `
- Fn + Esc: Esc
- Fn + H: Move Left
- Fn + J: Move Down
- Fn + K: Move Up
- Fn + I: Home
