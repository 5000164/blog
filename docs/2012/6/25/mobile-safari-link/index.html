Mobile Safari で数字に勝手にリンクが貼られるのを無効化する

Published 2012.6.25 by 菅原 浩

Mobile Safari では数字を電話番号だとみなして、勝手に電話番号のリンクを貼ることがあります。 これを防ぐためには勝手にリンクを貼る機能を無効化し、 telto: で明示的にリンクを貼りましょう。 機能を無効化するには、ヘッダに下記のmetaタグを追加します。

1

<meta name="format-detection" content="telephone=no">

telto: でのリンクの貼り方は下記になります。

1

<a href="telto:000-0000-0000">000-0000-0000</a>
