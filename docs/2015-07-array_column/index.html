PHP の二次元配列から同じキーの値だけで新しい配列を生成する

PHP の二次元配列から同じキーの値だけで新しい配列を生成する関数 array_column

他の人のコードを読んでたらたまたま見つけた便利な関数 array_column。
二次元配列から同じキーの値だけで新しい配列を生成する。
PHP: array_column - Manual

同じキーの値だけで新しい配列を作るサンプル

 1<?php
 2$data = [
 3    [
 4        'id' => 1,
 5        'name' => 'aaa',
 6    ],
 7    [
 8        'id' => 2,
 9        'name' => 'bbb',
10    ],
11];
12
13$id_list = array_column($data, 'id');
14$name_list = array_column($data, 'name');
15
16var_dump($id_list);
17// array(2) {
18//   [0]=>
19//   int(1)
20//   [1]=>
21//   int(2)
22// }
23
24var_dump($name_list);
25// array(2) {
26//   [0]=>
27//   string(3) "aaa"
28//   [1]=>
29//   string(3) "bbb"
30// }

二次元配列の縦横を入れ替える (転置行列を得る) サンプル

 1<?php
 2$data = [
 3    [
 4        'id' => 1,
 5        'name' => 'aaa',
 6    ],
 7    [
 8        'id' => 2,
 9        'name' => 'bbb',
10    ],
11];
12
13$transposed_matrix = [];
14foreach ($data[0] as $key => $value) {
15    $transposed_matrix[$key] = array_column($data, $key);
16}
17
18var_dump($transposed_matrix);
19// array(2) {
20//     ["id"]=>
21//   array(2) {
22//         [0]=>
23//     int(1)
24//     [1]=>
25//     int(2)
26//   }
27//   ["name"]=>
28//   array(2) {
29//         [0]=>
30//     string(3) "aaa"
31//     [1]=>
32//     string(3) "bbb"
33//   }
34// }
