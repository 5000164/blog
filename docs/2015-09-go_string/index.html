Go 言語で文字列としてダブルコーテーションや改行を入れる

golang でダブルコーテーションや改行を string 型の変数に入れる

go では “ (ダブルコーテーション) で文字列を表します。
文字列の中にダブルコーテーションを入れるには \ (バックスラッシュ) でエスケープすればよかった。

1
test := "test\"test"

改行を入れるには \n で入れられる。

1
newline := "a\nb\nc"

また、` (バッククオート) を使用することでヒアドキュメントのように書けた。

1
2
3
here := `like
a
here document`
