iOSでの文字入力

iOSでテキストボックスに文字を入力しますよね。

カーソルを移動する矢印がないですよね。

不便ですよね。

文字を右寄せにしているともっと不便です。

右端までなかなか移動できません。

どうして右寄せにしているのか

全部数字だったりすると桁を合わせて見やすくするためにも右寄せにしたりしますよね。

それが入力項目だから使いづらくなってるわけです。

iOSでも入力しやすくする方法

ひらめきました。

focusがあたっている時は左寄せにしてあげればいいのです。
